Puzzle Orbit

A circular puzzle game built in Unity where players rotate planetary rings, align constellations, and restore a broken sky map.

I focused on tactile rotation controls, subtle haptic-inspired animation, and a clean HUD that stays readable against dynamic backgrounds.

  • Role: UI/UX, front-end implementation
  • Platforms: PC prototype
  • Tags: Unity, Puzzle, Game Feel

Signal Echo

A narrative terminal where players scrub through noisy transmissions to reconstruct the story of a missing crew.

I designed the interface to feel like a real tool: scopes, dials, and readouts that animate with purpose, while remaining keyboard-accessible.

  • Role: UI design, interaction design
  • Focus: Narrative pacing, legibility
  • Tags: Unity, Narrative, UI Design

Glowline

A rhythm rail where players sketch paths for a glowing cursor, catching beats that travel along the track.

I focused on color contrast, timing feedback, and accessibility options like beat previews and customizable palettes.

  • Role: UI/UX, prototyping
  • Focus: Accessibility, game feel
  • Tags: Unity, Rhythm, Accessibility

Atlas Archive

A map-driven exploration game where every UI element exists as part of an in-world archivist desk.

I built layered localization-friendly layouts, ensuring that text expansion never breaks the illusion of the desk space.

  • Role: UI systems, layout
  • Focus: Diegetic UI, localization
  • Tags: Unity, Localization, Diegetic UI

Relay

A co-op puzzle where two players pass energy between devices in tight time windows, coordinating through UI signals.

I designed a clear signaling language using color, motion, and sound hooks that communicates urgency without overwhelming players.

  • Role: UI/UX, systems design
  • Focus: Co-op readability
  • Tags: Unity, Co-op, UI Systems
